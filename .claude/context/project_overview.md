# Project Overview: SEO Rank Tracker

## Product

**SEO Rank Tracker** is a professional SaaS rank-tracking tool built on the Google Search Console API. It provides keyword discovery, position tracking, intent classification, and page analytics — with no scrapers or proxies (100% Google API data).

**Live sites:**
- Marketing: https://www.seoranktrackingtool.com
- App: https://app.seoranktrackingtool.com

---

## Value Proposition

> "Clearer Rankings. Faster Reports. Smarter SEO."

- Auto-discover 10,000+ ranking keywords from GSC
- Intent classification (Informational / Transactional / Commercial / Navigational)
- Historical precision: daily position changes, 90-day history
- Global tracking: 150+ countries
- Privacy-compliant (GDPR & CCPA), no bot scrapers

- Single website owners
